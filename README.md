# HRIS UAE Demo (Engineering-grade internal demo)

## What this is
A self-contained React prototype for a Dubai/UAE HRIS based on your BRD:
- Sidebar navigation
- Organization (Mainland vs Free Zone)
- Branches
- Masters (Departments/Designations/Visa Types)
- Employees (Directory/Create/Profile)
- Onboarding checklist
- Documents
- Shifts + Roster
- Attendance (requests + reports + payroll-input summary)
- Leave
- Overtime
- Approvals
- Reports
- Security
- Settings

## Run locally
1) Install dependencies
```bash
npm install
```

2) Start dev server
```bash
npm run dev
```

## Notes
- No Tailwind, no shadcn, no path aliases.
- Uses lucide-react for icons.
